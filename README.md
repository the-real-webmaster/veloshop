# 🚲 VeloSHOP — Адаптивная верстка лендинга

Адаптивная HTML/CSS верстка лендинга‑магазина велосипедов с каталогом товаров, формой обратной связи и акцентом на удобную навигацию. Используются CSS Grid, Flexbox и адаптивная верстка для корректного отображения на различных устройствах.

## 🔗 Ссылка
[Посмотреть проект](https://the-real-webmaster.github.io/veloshop/)

## 🛠 Технологии
- HTML5  
- CSS3 (Flexbox, Grid, адаптивный дизайн)

## ✨ Основные особенности
- Адаптивная верстка под разные экраны  
- Каталог товаров в сетке  
- Форма обратной связи  
- Чёткая стилизация кнопок  
- Меню навигации
